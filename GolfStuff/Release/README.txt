========================================
  Birdie Mod v1.0 - Super Battle Golf
========================================

REQUIREMENTS
  - Super Battle Golf installed via Steam
  - Windows 10 or later

INSTALLATION
  1. Run BirdieModInstaller.exe
  2. The installer will automatically detect your game folder.
     If it is not found, click Browse and select the folder
     that contains "Super Battle Golf.exe".
  3. Click Install and wait for it to complete.
  4. Launch the game normally through Steam.

  Note: Windows Defender or other antivirus software may warn
  about the installer. This is a false positive caused by
  MelonLoader's DLL injection technique used by all game mods.
  The mod is safe to install.

UPDATING
  To update to a newer version, simply run the new installer and
  click Install again. It will automatically detect the existing
  installation and overwrite it with the updated files. You do
  not need to uninstall first.

UNINSTALLATION
  Run BirdieModUninstaller.exe and click Uninstall.
  Your original game files will be restored if a backup exists.

========================================
  MULTIPLAYER & HOST COMPATIBILITY
========================================

  Birdie Mod only affects the player who has it installed.
  Other players in your lobby are never affected unless noted.

  IF YOU ARE THE HOST:
    All features work at full strength. Additionally, Expanded
    Slots adds real server-side inventory slots — meaning every
    player in the lobby benefits from the expanded hotbar, even
    those without the mod installed.

  IF YOU ARE A CLIENT (not the host):
    All personal features still work. Expanded Slots gives you
    the visual UI expansion but extra slots have no server
    backing, so items cannot be placed in them without the host
    also running the mod.

========================================
  FEATURES
========================================

  Open the settings menu in-game with F6 to toggle features.

  [EVERYONE] Works fully for any player regardless of host status
  ---------------------------------------------------------------
  Swing Assist      - Auto-aims and releases your swing perfectly,
                      accounting for wind and distance to the hole
  Nearest Ball Mode - Target the closest ball on the course
  Perfect Shot      - Forces every shot to register as perfect
  No Knockback      - Ignore knockback from hits
  Ice Immunity      - Walk on ice without slipping
  Landmine Immunity - Walk over landmines without being hit
  Lock-On Any Dist. - Lock-on targets golf balls at any range
  Unlock Cosmetics  - Unlock all cosmetic items
  Coffee Boost      - Instant speed burst
  Infinite Ammo     - Never run out of item ammo
  HUD               - On-screen display showing active features

  [HOST RECOMMENDED] Full effect as host; may be partial as client
  ---------------------------------------------------------------
  No Wind           - Wind does not affect your ball trajectory
  No Air Drag       - Ball travels further through the air
  Speed Multiplier  - Move faster around the course
  No Recoil         - Eliminates weapon recoil

  [HOST PROVIDES FOR ALL] Host must have mod for full benefit
  ---------------------------------------------------------------
  Expanded Slots    - Expands hotbar to 8 total slots (keys 1-8).
                      As host: all players get real server slots.
                      As client: visual UI only for yourself.

========================================
  DEFAULT KEYBINDS
========================================

  F6  - Open/close settings menu
  F1  - Toggle swing assist
  F2  - Coffee boost
  F3  - Nearest ball mode
  F4  - Unlock cosmetics
  H   - Toggle HUD
  G   - Collect random item
  I   - Toggle ice immunity
  W   - Toggle no wind
  P   - Toggle perfect shot
  D   - Toggle no air drag
  S   - Toggle speed multiplier
  A   - Toggle infinite ammo
  R   - Toggle no recoil
  K   - Toggle no knockback
  M   - Toggle landmine immunity
  L   - Toggle lock-on any distance
  U   - Toggle expanded item slots

  All keybinds can be rebound in the Keys tab of the settings menu.
